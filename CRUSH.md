# Local Ollama Development Environment

## Primary LLM

- **Large/Medium**: `gemma4:26b-devops`
- **Small/Quick**: `devstral-small-2-gpu`
- **Provider**: Ollama Local at <http://localhost:11434/v1/>
- **Embeddings**: `nomic-embed-text`
- **Qdrant**: <http://localhost:6333>

## Specialization

This environment is optimized for DevOps, Terraform, Ansible, YAML, JSON, TypeScript/JavaScript/Node.js, Go, Python, and Rust coding workflows.
